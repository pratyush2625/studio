# **App Name**: LearnAI

## Core Features:

- Personalized Learning Paths: AI recommends personalized learning paths based on student's goals and current skill level, using Gemini API as a tool.
- Course Affiliate Integration: Integrate with platforms like Udemy and Coursera to suggest relevant courses and track affiliate commissions.
- Sentiment Analysis: Analyze student's learning emotions and engagement through sentiment analysis of their interactions.
- AI Mentor Chatbot: Provide real-time career guidance and mentorship through an AI-powered chatbot using Gemini API as a tool.
- Skill Portfolio: Enable students to showcase their skills through portfolios and find internships/projects.
- Interactive Skill Graph: Visualize skills and learning paths in an interactive graph format.
- User Authentication and Profiles: Secure user accounts with profile management and personalized settings.

## Style Guidelines:

- Primary color: Light, desaturated blue (#B0E2FF) for a calm and trustworthy learning environment.
- Background color: Very light blue (#F0F8FF).
- Accent color: Soft green (#A0D6B4) to highlight progress and positive feedback.
- Body and headline font: 'PT Sans' for a balance of modernity and readability.
- Use clean, modern icons to represent different skills and learning resources.
- Implement a clean, intuitive layout with clear information hierarchy and easy navigation.
- Use subtle animations to provide feedback and guide users through the platform.
